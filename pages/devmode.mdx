O Dev mode é o ambiente de desenvolvimento da V4Pay, projetado para permitir que você teste todas as funcionalidades da plataforma de forma segura e controlada.

## <Icon icon="flask" type="solid"/>  O que é o Dev mode?

Quando você cria sua conta na V4Pay, ela é automaticamente configurada em Dev mode. Neste ambiente:

- Todas as operações são simuladas
- Nenhuma transação real é processada
- Você pode testar todas as funcionalidades sem riscos
- Os dados são isolados do ambiente de produção

<Card title="Benefícios do Dev mode" icon="star" horizontal>
  * Teste suas integrações com segurança
  * Experimente diferentes cenários de pagamento
  * Configure webhooks e receba notificações de teste
  * Valide o fluxo completo da sua aplicação
</Card>

## <Icon icon="key" type="solid"/>  Usando o Dev mode

Para começar a testar:

1. Crie uma chave de API no ambiente de desenvolvimento
2. Use esta chave em suas requisições de teste
3. Configure webhooks para receber notificações de teste
4. Valide todos os fluxos da sua aplicação

<Card title="Dicas para Testes Efetivos" icon="lightbulb" horizontal>
  * Teste todos os cenários possíveis
  * Valide o tratamento de erros
  * Verifique as notificações de webhook
  * Confirme a integração com seu sistema
</Card>

## <Icon icon="arrow-right" type="solid"/>  Próximos Passos

Quando estiver confiante com sua integração:
1. Desative o Dev mode
2. Complete o processo de verificação da conta
3. Aguarde a aprovação para produção
4. Comece a processar transações reais
